Installation

To run the web application Node.js must be installed on your machine

Follow the steps below to install the application

  • npx create-react-app "destination_folder" or yarn create react-app "destination_folder" (If yarn is installed)
  • npm install react-router-dom framer-motion or yarn add react-router-dom framer-motion (If yarn is installed)

To run the application locally

  • npm start or yarn start (If yarn is installed)

To create a deloyable folder

  • npm build or yarn build (If yarn is installed)
