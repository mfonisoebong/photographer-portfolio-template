All components and routes are located in .src/components and .src/routes folder

When you open up the folder in your text editor or any IDE of your choice. The folders as well as the components and routes are all enclosed in their respective folders as shown below

Edit Header

To edit header check folder src components Navbar.js

Edit Homepage

To edit landing page details check folder src routes Homepage.js

You may decide to change the links in the component by replacing the href value to an actual link

Example:
href="https://facebook.com/myprofile"

Edit Skills page

To edit skills page details check folder src routes Skills.js

It is recommended that you change the images by going into the src images folder to change the images

Edit Resume/CV page

To edit Resume/CV page details check folder src routes Resume.js

It is recommended that you change the images by going into the src images folder to change the images

To change the default download resume file, you can check folder src resume and replace the file

Then move to src routes Resume.js and change the file directory

Image resetting

All images used for the original template product may be replaced or left as it is.
This solely depends on the choice of the party (or parties) in ownership of this product.

All images are located in the "./src/images" folder

It is recommended that Images used for this template should be of 600 by 430 dimension

Installation

To run the web application Node.js must be installed on your machine.

To install NodeJs, go to https://nodejs.org/en/download/ to download and install NodeJs on your machine

Follow the steps below to install the application from the terminal

  • npx create-react-app "destination_folder" or yarn create react-app "destination_folder" (If yarn is installed)
  • npm install react-router-dom framer-motion or yarn add react-router-dom framer-motion (If yarn is installed)

To run the application locally

  • npm start or yarn start (If yarn is installed)

To create a deloyable folder

  • npm build or yarn build (If yarn is installed)
